# 🚨 Enable GitHub Pages - Step by Step

## The page should be open in your browser now!

If not, go to: https://github.com/Vedant294/exam-form-submission/settings/pages

---

## 📋 Follow These Exact Steps:

### 1. Look for "Build and deployment" section

### 2. Under "Source":
- Click the dropdown (it says "None" or "Deploy from a branch")
- Select: **Deploy from a branch**

### 3. Under "Branch":
- First dropdown: Select **gh-pages** (NOT main)
- Second dropdown: Select **/ (root)**
- Click **Save** button

### 4. Wait for Confirmation
You'll see a blue box saying:
"Your site is ready to be published at https://vedant294.github.io/exam-form-submission/"

---

## ⏱️ Wait 2-3 Minutes

After enabling, GitHub needs time to build and deploy.

Then visit: **https://vedant294.github.io/exam-form-submission**

---

## 🔍 Troubleshooting

### If you don't see "gh-pages" in the dropdown:
The branch exists! Try:
1. Refresh the page
2. Check "Branches" tab: https://github.com/Vedant294/exam-form-submission/branches
3. You should see "gh-pages" branch there

### If still 404 after 5 minutes:
1. Go to Actions tab: https://github.com/Vedant294/exam-form-submission/actions
2. Check if "pages build and deployment" is running
3. Wait for it to complete (green checkmark)

---

## ✅ Success Indicators

When it's working, you'll see:
- Green checkmark in Actions tab
- "Your site is live at..." message in Pages settings
- Live demo works at the URL
